Описание решение по файлам:

Самый главный файл - cli.py.
  Вызывая функцию console_emulation() - появляется консоль.
  console_emulation содержит цикл while true, 
  внутри которого сплитится входная строка и проверятся, есть ли в ней |. 
  Если пайпа нет, то вызывавется функция execute_command,
  если есть, то вызывается функция pipe_execution
  (обеим передается list, содержащий входные данные).

  pipe_execution нахоит индексы | и разделяет весь лист на подкоманды.
  Дальше последовательно каждая подкоманда выполняется.

Файл IO_classes
  Для того, чтобы соединить IO команд, разделенных пайпом,
  в файле IO_classes введены два класса: один содержит данные,
  второй - выдает данные. Метод convert_to_input класса output_stream 
  связывает два этих класса.

Файл utilities
  В файле utilities находятся реализации всех команд, которые нужно было
  сделать. Так же есть функция not_implemented_finctions, которая
  вызывается в случае, если введеная в "консоль" команда не реализована.

Файл decorators
  Содержит два декоратора.
  Первый позволяет вызывать функцию с аргументами по имени.
  Второй - выводит результат выполненной команды на экран.
  
Файл aux_functions
  Содержит две небольшие функции, которые парсят входную строку.
  Первая вызывается, если во входное строке был найден "$",
  вторая - если "=".
